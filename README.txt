Walmart Weekly Sales Analysis

This project explores seasonal and economic factors that impact weekly
sales performance across Walmart stores.

Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

Dataset
Kaggle Walmart Sales Dataset

Key Questions
1. Do holiday weeks impact sales?
2. Which stores generate the highest average sales?
3. Which months perform best?
4. Does unemployment affect sales?
5. Are seasonal patterns consistent across years?
6. How consistent are sales across stores?

Key Insights
- Holiday weeks generate ~7.5% higher sales
- Strong seasonal patterns exist with peaks in Nov–Dec
- Higher unemployment correlates with lower weekly sales
- Store performance varies significantly across locations
